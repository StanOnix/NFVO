== VNFDependencies

=== Adding a VNFDependency

AVNFDependency is added by using a `POST` request like this:

'/api/v1/ns-descriptors/{nsdId}/vnfdependencies/'

The VNFDependency should be in the request's body.


=== Retrieving all VNFDependency

One can det all AVNFDependencies by using a `GET` request like this:

'/api/v1/ns-descriptors/{nsdId}/vnfdependencies/'


=== Retrieving a VNFDependency

A specific AVNFDependency can be retrieved by using a `GET` request like this:

'/api/v1/ns-descriptors/{nsdId}/vnfdependencies/{vnfdependencyId}'


=== Updating a VNFDependency

AVNFDependency is updated by using a `PUT` request like this:

'/api/v1/ns-descriptors/{nsdId}/vnfdependencies/{vnfdependencyId}'

The new VNFDependency should be in the request's body.


=== Removing a VNFDependency

AVNFDependency is removed by using a `DELETE` request like this:

'/api/v1/ns-descriptors/{nsdId}/vnfdependencies/{vnfdependencyId}'
